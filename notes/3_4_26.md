Earth Engine Recap
---
- Feature
    - Line/point/polygon
    - List of properties (metadata)
- Image
    - Stack of georeferenced bands
    - Each band has its own:
        - Mask, projection, and resolution
        - In Earth Engine world resolution = scale
    - List of properties (metadata)
        - Date, bounding box, footprint, cloud cover, etc.
- Collection
    - Pretty much just a bag of elements
    - Can have a collection of images and/or features
        - Table of features
        - List of images
    - Can filter, sort, join, map, reduce
---
- Map:
    - Apply a function to each dataset of a collection
    - Way of doing for-loop style calculations
    - "For each thing in this collection, do this calculation"
    - Examples: 
        - Area of each feature
        - Cloud covers of each image (over a region of interest)
        - Mosaic for each month

- Reduce:
    - Apply a function to **everything** in a collection
    - This is the "aggregation operation"
    - Examples:
        - Sum area over all features
        - Calculate a mean/median composite
        - Train a classifier
